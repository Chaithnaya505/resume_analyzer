# Resume Analyzer

An intelligent Resume Analyzer built using Python, NLP, and Machine Learning to compare resumes with job descriptions and calculate matching scores.

---

## Features

- Resume and Job Description Matching
- Skill Extraction and Analysis
- Match Score Prediction
- NLP-Based Text Processing
- CSV Dataset Integration
- Simple and User-Friendly Implementation

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Natural Language Processing (NLP)
- Machine Learning

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/resume-analyzer.git
```

Navigate to the project folder:

```bash
cd resume-analyzer
```

Install required packages:

```bash
pip install pandas scikit-learn numpy
```

---

## Run the Project

```bash
python app.py
```

---

## How It Works

1. Reads resume and job description data
2. Processes text using NLP techniques
3. Compares skills and keywords
4. Calculates similarity/match score
5. Displays matching results
